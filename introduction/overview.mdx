Whether you're a beginner or an experienced developer looking for a streamlined deployment workflow, this documentation covers everything you need to get your application live on the web.

## Prerequisites

Before you begin, make sure you have the following:

<Steps>
  <Step title='Node.js (v18.18 or later)'>
    Download and install Node.js from [nodejs.org](https://nodejs.org). Verify your installation:
    ```bash node --version # Should output v18.18.0 or later ```
  </Step>
  <Step title='A GitHub Account'>
    Sign up for a free account at [github.com](https://github.com) if you don't already have one.
  </Step>
  <Step title='A Vercel Account'>
    Create a free Vercel account at [vercel.com](https://vercel.com). We recommend signing up with
    your GitHub account for seamless integration.
  </Step>
  <Step title='A Code Editor'>
    We recommend [Visual Studio Code](https://code.visualstudio.com/) with the following extensions:
    - **ESLint** — for code linting - **Prettier** — for code formatting - **Tailwind CSS
    IntelliSense** — for Tailwind autocompletion
  </Step>
</Steps>

## Guide Structure

This guide is organized into **five sequential chapters**. Follow them in order for the best experience:

<Steps>
  <Step title='Creating a Next.js Application'>
    Scaffold a new project using `create-next-app` and explore the project structure.
  </Step>
  <Step title='Setting Up a GitHub Repository'>
    Version-control your code and push it to a remote GitHub repository.
  </Step>
  <Step title='Connecting the Project to Vercel'>
    Link your GitHub repo to Vercel for automatic build and deployment.
  </Step>
  <Step title='Managing Environment Variables'>
    Configure secrets and environment-specific values for your app.
  </Step>
  <Step title='Deployment Steps'>
    Deploy to production, set up custom domains, and monitor your application.
  </Step>
</Steps>

<Note>
  Each chapter includes code snippets, terminal commands, and screenshots to help you follow along.
  You can complete this entire guide in **under 30 minutes**.
</Note>
